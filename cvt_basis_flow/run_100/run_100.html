RUN_100
4 CVT Basis Vectors
for BURGERS Data

This run works with the 40 scalar flow solutions from the BURGERS dataset. We don't use an XY file, we don't use a steady state file, we don't do any preprocessing, and we don't do mass matrix preconditioning.

  • cvt_run_100.inp, an input file that computes a CVT basis of size 4.
  • cvt_run_100.out, the output of the run.
  • gen_001.txt, a CVT basis vector.
  • gen_002.txt, a CVT basis vector.
  • gen_003.txt, a CVT basis vector.
  • gen_004.txt, a CVT basis vector.
