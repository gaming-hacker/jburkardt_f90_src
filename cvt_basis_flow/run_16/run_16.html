RUN_16
4 CVT Basis Vectors
for TCELL Data

Run #16 works with 500 flow solutions in the T-Cell region. We subtract 5/3 of steady solution from 1-250, and 1/3 from 251 through 500. We NORMALIZE the results.

  • cvt_run_16.inp, an input file that computes the energy of a set of preprocessed data for cluster size 4.
  • cvt_run_16.out, the output of the run.
  • gen_001.txt, centroid #1.
  • gen_002.txt, centroid #2.
  • gen_003.txt, centroid #3.
  • gen_004.txt, centroid #4.
