RUN_37 - A run of CVT_BASIS

Run #37 works with 500 flow solutions in the CAVITY region. We subtract 5/3 of steady solution from 1-250, and 1/3 from 251 through 500. We DON'T normalize the PDE solutions.

  • cvt_run_37.inp, an input file that computes the energy of a set of preprocessed data for cluster size 2.
  • cvt_run_37.out, the output of the run.
  • gen_001.txt, centroid #1.
  • gen_002.txt, centroid #2.
