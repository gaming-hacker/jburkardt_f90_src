F90_SIMPLE
A Simple Makefile Example

F90_SIMPLE is a simple example of how a makefile can be used to manage a set of FORTRAN90 files.

The underlying task is to approximate the integral of a function F(T) using the midpoint rule. The program to do this is broken up into three FORTRAN90 files:

  • f90_simple.f90 contains the main program, which tries the midpoint rule three times, each time using more intervals.
  • midpoint.f90 carries out the midpoint rule to approximate the integral of a function.
  • f.f90 evaluates the function whose approximate integral is desired.

  • makefile contains the makefile for managing these files.
  • f90_simple_output.txt the output file.
