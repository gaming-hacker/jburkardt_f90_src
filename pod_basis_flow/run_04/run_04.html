RUN_04
A run of POD_BASIS

This run of POD_BASIS takes the CAVITY_FLOW data, applies mass matrix preconditioning, and seeks 16 POD vectors.

  • run_04.inp, the input file to POD_BASIS;
  • run_04.out, the output of the run.
